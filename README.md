# Firdaus Portfolio — Vite + React + Tailwind

## Setup

```bash
npm install
npm run dev
```

## Notes

- Video background is at `public/bg-video.mp4` (already included)
- Font: **Libre Baskerville** (serif display) + **DM Sans** (body) — loaded via Google Fonts
- Animations: Framer Motion
- To update social links or text, edit `src/components/Hero.jsx`, `Navbar.jsx`, `BottomBar.jsx`

## Build

```bash
npm run build
```
